# Data Analytics Project 4 - Data Visualization

## Overview

This project focuses on transforming raw data into meaningful visual insights through effective data visualization techniques.

## Objective

The goal of this project is to create clear and informative charts that help users understand trends, comparisons, and distributions within a dataset.

## Visualizations Included

* Bar Chart: Sales Comparison Across Products
* Line Chart: Monthly Growth Trend
* Pie Chart: Market Share Distribution

## Key Insights

* Mango recorded the highest sales among all products.
* Monthly growth showed a consistent upward trend.
* Company A held the largest market share compared to competitors.

## Skills Demonstrated

* Data Visualization
* Data Storytelling
* Insight Generation
* Chart Selection
* Business Communication

## Tools Used

* Canva
* PDF Export
* GitHub

## Conclusion

This project demonstrates how visual representations of data can simplify complex information and support better decision-making.
