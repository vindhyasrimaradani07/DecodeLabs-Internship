# Interactive Web Elements

## Overview

This project demonstrates interactive web elements using HTML, CSS, and JavaScript.

## Features

* Dark Mode Toggle
* Dynamic webpage interaction
* User-friendly interface
* Responsive design

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Open the project folder.
2. Open index.html in a web browser.
3. Click the "Toggle Dark Mode" button.
4. The page will switch between light and dark themes.

## Project Functionality

The website allows users to toggle between light mode and dark mode using JavaScript.
